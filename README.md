# IncrementGameUnity

## Features & Keybinds

- This is a 1 player game
- The rules are simple, just tap as much as you can to upgrade your resources to be rich man!.
- Controls:
  - Left click to coin area
  
## Additional Features
- Nice design color UI
- Sfx sounds while :
  - Unlocked new resources!
  - Tapping coins
